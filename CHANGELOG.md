## [v0.2.0] - 2026-07-16

### Highlights

- New `asset` package for uploading audio, images, and video to HeyGen's asset service (upload.heygen.com)

### Added

- `asset.Client` with `Upload` returning a hosted URL usable in other HeyGen APIs (e.g., audio-driven video generation) ([`875ba80`](https://github.com/plexusone/heygen-go/commit/875ba80))
- `asset.ContentType*` constants for the supported asset types (JPEG, PNG, MP4, WebM, MPEG audio) ([`875ba80`](https://github.com/plexusone/heygen-go/commit/875ba80))
- `Client.Asset` facade field on the root SDK client ([`875ba80`](https://github.com/plexusone/heygen-go/commit/875ba80))
- `heygen.Client.RequestURL` for raw-body requests against absolute URLs (endpoints hosted outside the API base URL) ([`5d45e9e`](https://github.com/plexusone/heygen-go/commit/5d45e9e))
- `avatar.ListV2` and `avatar.SearchV2` for the v2 avatars API, returning IDs directly usable as avatar_id in generation (unlike the v3 avatar-group IDs) ([`991df25`](https://github.com/plexusone/heygen-go/commit/991df25))
- `omniavatar` subpackage: an OmniAvatar render adapter implementing the omniavatar-core render interfaces (`render.Provider`, `render.AudioUploader`, `render.AvatarLister`) on top of this SDK; depends only on omniavatar-core ([`d52d734`](https://github.com/plexusone/heygen-go/commit/d52d734))
- Unit tests for the asset upload client ([`875ba80`](https://github.com/plexusone/heygen-go/commit/875ba80))
- Unit tests for `RequestURL` and retry body rewind ([`5d45e9e`](https://github.com/plexusone/heygen-go/commit/5d45e9e))

### Fixed

- Retries now rewind the request body via `GetBody`, so a 429-retried POST resends the full payload instead of an empty body ([`47e28bf`](https://github.com/plexusone/heygen-go/commit/47e28bf))

### Documentation

- README: asset upload feature, usage example, and API coverage row ([`dd20e78`](https://github.com/plexusone/heygen-go/commit/dd20e78))
- MkDocs site: Asset API page, nav entry, and index/overview updates ([`dd20e78`](https://github.com/plexusone/heygen-go/commit/dd20e78))
- Adopt the shared PlexusOne MkDocs theme (custom palette, unified navigation override) ([`3287c7b`](https://github.com/plexusone/heygen-go/commit/3287c7b))

## [v0.1.0] - 2026-07-12

### Highlights

- Initial release of the HeyGen Go SDK with Avatar and LiveAvatar API support

- Main SDK entry point with `heygen.New()` constructor ([`3367459`](https://github.com/plexusone/heygen-go/commit/3367459))
- Avatar v3 API client: `List`, `Get`, `ListLooks` ([`593b00c`](https://github.com/plexusone/heygen-go/commit/593b00c))
- LiveAvatar real-time streaming client: `NewSession`, `StartSession`, `StopSession` ([`071ff4e`](https://github.com/plexusone/heygen-go/commit/071ff4e))
- LITE mode support for BYO AI stack with LiveKit integration ([`071ff4e`](https://github.com/plexusone/heygen-go/commit/071ff4e))
