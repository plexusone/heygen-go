## [v0.2.0] - 2026-07-16

### Highlights

- New `asset` package for uploading audio, images, and video to HeyGen's asset service (upload.heygen.com)

### Added

- `asset.Client` with `Upload` returning a hosted URL usable in other HeyGen APIs (e.g., audio-driven video generation)
- `asset.ContentType*` constants for the supported asset types (JPEG, PNG, MP4, WebM, MPEG audio)
- `Client.Asset` facade field on the root SDK client
- `heygen.Client.RequestURL` for raw-body requests against absolute URLs (endpoints hosted outside the API base URL)
- Unit tests for asset upload and retry body handling

### Fixed

- Retries now rewind the request body via `GetBody`, so a 429-retried POST resends the full payload instead of an empty body

### Documentation

- README: asset upload feature, usage example, and API coverage row
